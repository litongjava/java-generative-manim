        -   斜坡上放置一个矩形代表雪橇。
        -   用箭头指示雪橇沿斜坡向下的位移方向，并标注 \(d = 30.0\ \text{m}\)。

### 【场景三：应用物理定律与力值计算】

-   **背景与布局**
    -   背景保持浅灰色或白色。
    -   右下角显示场景编号“03”。
    -   布局可考虑左右分栏：左侧展示理论推导，右侧显示力的计算结果。

-   **内容呈现**
    -   **左侧（理论推导）**：
        -   标题：“2. 依据“恒速运动”计算力值”，`Write` 效果。
        -   核心条件文本：“雪橇 **恒速** 下滑 \(\implies\) 加速度 \(a=0\)，沿斜面方向合力为零。” `FadeIn`。
        -   列出沿 x 轴的平衡方程（使用 LaTeX）：
            \[
            \sum F_x = mg\sin\theta - f - T = 0
            \]
            公式使用 `Write` 或 `FadeIn` 效果。
        -   推导绳子拉力 T 的表达式：
            \[
            T = mg\sin\theta - f
            \]
        -   补充摩擦力公式：
            \[
            f = \mu N
            \]
        -   补充支持力 N 的计算（由 y 轴方向平衡得到）：
            \[
            \sum F_y = N - mg\cos\theta = 0 \implies N = mg\cos\theta
            \]
        -   将 N 代入 f，再代入 T 的表达式：
            \[
            T = mg\sin\theta - \mu mg\cos\theta = mg(\sin\theta - \mu\cos\theta)
            \]
            使用 `ReplacementTransform` 或逐步 `Write` 展示推导过程。

    -   **右侧（数值计算）**：
        -   标题：“力的数值计算 (g=9.8 m/s²)”，`FadeIn`。
        -   依次计算并展示：
            -   \(mg\sin60^\circ = 90.0 \times 9.8 \times \sin60^\circ \approx 762.9\ \text{N}\)
            -   \(N = mg\cos60^\circ = 90.0 \times 9.8 \times \cos60^\circ = 441.0\ \text{N}\)
            -   \(f = \mu N = 0.100 \times 441.0 = 44.1\ \text{N}\)
            -   \(T = mg\sin\theta - f = 762.9 - 44.1 = 718.8\ \text{N}\)
        -   每个计算结果使用 `FadeIn` 或 `Write` 效果，并用矩形框 (`SurroundingRectangle`) 短暂高亮。

-   **相机与过渡效果**
    -   相机稳定，确保左右两栏内容清晰可见。若内容过多，可先聚焦左侧推导，再平移至右侧计算。
    -   公式和计算结果的出现节奏适中，便于观众跟上思路。

### 【场景四：计算各力做的功】

-   **背景与布局**
    -   背景保持浅灰色或白色。
    -   右下角显示场景编号“04”。
    -   屏幕主体用于展示功的计算过程。

-   **内容呈现**
    -   **标题**：屏幕上方显示 “3. 计算各力所做的功”，`Write` 效果。
    -   **功的定义公式**：首先展示功的一般计算公式：
        \[
        W = F d \cos\phi
        \]
        其中 \(\phi\) 是力 F 与位移 d 的夹角。公式 `FadeIn`。
    -   **分项计算**：依次计算每个力做的功，每项计算包括：
        -   **子标题**（如“(a) 摩擦力做的功”），`FadeIn`。
        -   **说明**：简述力 \(F\)、位移 \(d\) 和夹角 \(\phi\) 的值。
        -   **计算公式与结果**（使用 LaTeX）：
            -   **摩擦力 (a)**：
                -   \(f = 44.1\ \text{N}\), \(d = 30.0\ \text{m}\), \(\phi = 180^\circ\) (\(\cos 180^\circ = -1\))
                -   \[ W_{\text{摩擦}} = f d \cos 180^\circ = (44.1)(30.0)(-1) = -1323\ \text{J} \approx -1.32 \times 10^3\ \text{J} \]
            -   **绳子拉力 (b)**：
                -   \(T = 718.8\ \text{N}\), \(d = 30.0\ \text{m}\), \(\phi = 180^\circ\)
                -   \[ W_{\text{绳子}} = T d \cos 180^\circ = (718.8)(30.0)(-1) = -21564\ \text{J} \approx -2.16 \times 10^4\ \text{J} \]
            -   **重力 (c)**：
                -   作用力为重力沿斜面的分量 \(mg\sin\theta = 762.9\ \text{N}\), \(d = 30.0\ \text{m}\), \(\phi = 0^\circ\) (\(\cos 0^\circ = 1\))
                -   \[ W_{\text{重力}} = (mg\sin\theta) d \cos 0^\circ = (762.9)(30.0)(1) = 22887\ \text{J} \approx 2.29 \times 10^4\ \text{J} \]
        -   每个计算步骤使用 `Write` 或 `FadeIn` 逐步展示。最终结果可以用不同颜色（如负功用红色 `RED`，正功用绿色 `GREEN`）或高亮框突出。

-   **相机与过渡效果**
    -   相机保持静止，聚焦于计算过程。
    -   每个力的功计算完成后，可以有一个短暂的停顿。

### 【场景五：计算总功与结论总结】

-   **背景与布局**
    -   背景可切换回初始的蓝白渐变或使用深色背景（如深蓝 `#0A192F`）以示总结。
    -   右下角显示场景编号“05”。
    -   内容居中或分点列出。

-   **内容呈现**
    -   **计算总功 (d)**：
        -   标题：“(d) 总功计算”，`FadeIn`。
        -   方法一：直接求和
            \[
            W_{\text{总}} = W_{\text{重力}} + W_{\text{摩擦}} + W_{\text{绳子}}
            \]
            \[
            W_{\text{总}} \approx 22887\ \text{J} + (-1323\ \text{J}) + (-21564\ \text{J}) = 0\ \text{J}
            \]
            展示加法过程，最终结果 `Write` 或 `FadeIn`。
        -   方法二：功能关系解释
            -   文本：“或根据功能关系：由于雪橇 **恒速** 运动，动能变化 \(\Delta E_k = 0\)。根据动能定理 \(W_{\text{总}} = \Delta E_k\)，可知：”
            -   \[ W_{\text{总}} = 0\ \text{J} \]
            这个解释可以作为补充说明 `FadeIn` 出现。

    -   **最终答案总结**：
        -   标题：“最终答案”，`Write` 效果，使用醒目颜色（如金色 `GOLD`）。
        -   使用 `BulletedList` 或 `VGroup` 垂直排列最终结果：
            -   (a) 摩擦力做的功： \(W_{\text{摩擦}} \approx -1.32 \times 10^3\ \text{J}\)
            -   (b) 绳子做的功： \(W_{\text{绳子}} \approx -2.16 \times 10^4\ \text{J}\)
            -   (c) 重力做的功： \(W_{\text{重力}} \approx 2.29 \times 10^4\ \text{J}\)
            -   (d) 总功： \(W_{\text{总}} = 0\ \text{J}\)
        -   结果列表使用 `FadeIn` 或 `Write` 逐条显示。

-   **相机与过渡效果**
    -   相机聚焦于总功计算，然后平移或缩放调整至最终答案总结。
    -   在显示最终结果时，镜头可以轻微放大（`self.camera.frame.scale(0.9)`）以示强调。
    -   场景结束前保持画面稳定几秒钟。
